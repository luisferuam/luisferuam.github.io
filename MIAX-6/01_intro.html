Ejemplo: Carga de datos y ejecución de una red neuronal sencilla

Como primera prueba, vamos a utilizar el código del libro de M. Nielsen (https://github.com/mnielsen/neural-networks-and-deep-learning ).

El código, adaptado para python 3, está disponible en la carpeta del curso, subcarpeta codigo-Nielsen. Haz una copia de esta carpeta en tu Google Drive.

Lo primero que hacemos es añadir la carpeta con el código al path. Si has seguido los pasos anteriores todo funcionará correctamente. Si no, revisa las rutas.

import sys
sys.path.append('miax-6/codigo-Nielsen/')

E importamos los módulos necesarios:

import mnist_loader
import network

A continuación cargamos los datos del problema:

datafile = "miax-6/codigo-Nielsen/data/mnist.pkl.gz"
trn_data, val_data, tst_data = mnist_loader.load_data_wrapper(datafile)
trn_data = list(trn_data)
val_data = list(val_data)
tst_data = list(tst_data)

Y reducimos el tamaño de los datos de entrenamiento:

trn_data = trn_data[:1000]

Creamos una red con 30 neuronas ocultas y la entrenamos 10 épocas:

net = network.Network([784, 30, 10])
net.SGD(trn_data, 10, 10, 3.0, test_data=tst_data)

Requisitos previos

Álgebra lineal:

  • Escalares, vectores, matrices, tensores.
  • Transposición.
  • Producto de matrices (dot product).

Ver capítulo 2 del libro de Goodfellow: https://www.deeplearningbook.org/contents/linear_algebra.html

Programación con python y NumPy:

  • NumPy arrays.
  • Indexación de arrays.
  • Matemática con arrays.
  • Broadcasting.

Ver tutorial en el curso de Stanford: https://cs231n.github.io/python-numpy-tutorial/#numpy

Ejercicios

Para ver cómo lleváis los conceptos básicos de álgebra lineal.

Sean los siguientes vectores y matrices:

\[ {\bf x} = (3, -2, 4)^{t},\;\;\; {\bf y} = (1, 5, 2)^{t},\;\;\; {\bf A} = \begin{pmatrix} 1 & 2 & 3\\ -3 & 1 & 2 \end{pmatrix},\;\;\; {\bf z} = (-1, 2)^{t} \]

Calcula:

  1. \({\bf x}^{t} {\bf y}\)

  2. \({\bf A} {\bf x}\)

  3. \({\bf A}^{t}\)

  4. \({\bf z}^{t} {\bf A}\)
